# Web5Layer Validator Starter Kit

Welcome to the Web5Layer validator node setup. This kit includes everything needed to launch a node and become a validator.

## Instructions

1. Upload `geth` binary to `validator_starter_kit/` (same folder as `start-node.sh`)
2. Run the script: `chmod +x start-node.sh && ./start-node.sh`
3. Follow prompts to create wallet and launch your node.
